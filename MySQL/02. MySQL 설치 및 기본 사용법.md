● 기본 사용법

- 로그인 후 프롬프트 상에서

| 명령어              | 설명                         |
| ------------------- | ---------------------------- |
| show databases;     | db리스트                     |
| use world;          | world db 사용                |
| show tables;        | world db의 테이블 리스트     |
| desc city;          | city 테이블의 구조를 표시    |
| select * from city; | city 테이블의 내용 모두 표시 |

 

- desc 명령어는 테이블 스키마(구조)를 표시하는 명령
